숫자 함수


ABS

절대값을 반환 합니다.


SIGN

0, 양수, 음수를 판별해 줍니다.
0은 0, 양수는 +1, 음수는 -1 반환


ROUND

지정된 자리수까지 반올림 합니다.
자리수 기본값은 0 입니다.
ROUND(숫자, 자리수)


TRUNC

지정된 자리수까지 버립니다.
자리수 기본값은 0 입니다.
TRUNC(숫자, 자리수)


CEIL

올림값을 반환 합니다.


FLOOR

내림값을 반환 합니다.


MOD

나눈 값의 나머지값을 반환 합니다.
(N2 * FLOOR(N1/N2)) - N1


REMAINDER

나눈 값의 나머지값을 반환 합니다.
(N2 * ROUND(N1/N2)) - N1


POWER

n의 제곱을 반환 합니다.


SQRT

제곱근을 반환 합니다.
