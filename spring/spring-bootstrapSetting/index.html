Bootstrap 설정하기

build.gradle 설정하기

mvnrepository에 접속한다.
검색란에 'bootstrap'를 검색한 후 org.webjars 패키지를 선택한다.
원하는 버전을 클릭 후 build.gradle에 추가해준다.
!! 이때 mvn페이지를 종료하지 않고 대기한다.
gradle -> Source Sets -> main 에 webjars가 있는지 확인한다.


jar 파일 입력하기

위의 mvnrepository 버전 클릭시 Files에서 jar파일을 다운로드 한다.
압축을 풀고 /META-INF/resources 내부의 webjars를 복사한다.
프로젝트의 /resources/static에 vendor폴더를 생성한다.
webjars 폴더를 vendor폴더에 넣어준다. ( resources/static/vendor/webjars )
아래 폴더 구조를 참고하자.


Bootstrap 경로 추가하기

html의 head태그에 link를 추가한다.
경로는 /vendor/webjars/bootstrap/css/bootstrap.css 가 된다.


확인하기

정상적으로 설정이 되었으면 화면에 파란 버튼이 생성된다.
