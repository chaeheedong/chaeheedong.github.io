Thymeleaf 파일 구조

필수 HTML 속성

Thymeleaf를 이용하려면 html 태그에 속성을 추가해야 한다.
속성 xmlns:th="http://thymeleaf.org" 추가





사용방법

페이지간 이동

th:href="@{path}"
페이지간 이동시 사용한다.
쿼리 스트링 전송이 가능하다.


서버 데이터를 화면에 텍스트로 출력

서버에서 Model을 이용하여 화면단에 데이터를 전송한다.
화면에서 th:text="${key}" 를 이용하여 출력한다.


서버 객체를 화면에 텍스트로 출력

서버에서 Model을 이용하여 화면단에 객체로 데이터를 전송한다.
화면에서 th:text="${객체.key}" 를 이용하여 출력한다.


서버 리스트 데이터를 화면에 반복문으로 출력

서버에서 Model을 이용하여 화면단에 리스트 객체로 데이터를 전송한다.
화면에서 th:each="별칭, status: ${객체}" 를 이용하여 받는다.
화면에서 th:text="${별칭.key}" 를 이용하여 반복 출력한다.
