카티시안 곱

카티시안 곱 이란?

테이블 n개를 Join 하지 않고 조회시 발생한다.
상단 예제 DDL 의 카티시안 곱 발생시 25개의 결과가 나온다.
tableA(5) * tableB(5) = 25
Join 조건이 누락 되었을 때 발생한다.





조인

등가 조인

Equi Join 으로도 불린다.
WHERE 절에 등호(=)를 이용하여 조인한다.
aliasA.column = aliasB.column
조인하는 양쪽의 컬럼값이 동일한 것만 조인에 성공한다.


비등가 조인

Non Equi Join 으로도 불린다.
WHERE 절에 등호(=) 외의 다른 조건을 이용하여 조인한다.
등호(=) 외에 >=, <=, >, <, != 등등 있다.
조인하는 조건이 동일한 것만 조인에 성공한다.


이너 조인

Inner Join 으로도 불린다.
수학의 교집합과 같다.
최상단 그림 예제의 중간에 해당한다.
FROM 절 아래에 INNER JOIN table alias ON condition 을 기술한다.


레프트 아우터 조인

Left Outer Join 으로도 불린다.
최상단 그림 예제의 좌측 상단에 해당한다.
FROM 절 아래에 LEFT JOIN table alias ON condition 을 기술한다.


레프트 아우터 조인 [교집합 제거]

Left Outer Join 에서 교집합을 제거한다.
최상단 그림 예제의 좌측 상단 두번째에 해당한다.
FROM 절 아래에 LEFT JOIN table alias ON condition 을 기술한다.
WHERE 절에 JOIN alias.column IS NOT NULL 을 기술한다.


라이트 아우터 조인

Right Outer Join 으로도 불린다.
최상단 그림 예제의 우측 상단에 해당한다.
FROM 절 아래에 RIGHT JOIN table alias ON condition 을 기술한다.


라이트 아우터 조인 [교집합 제거]

Right Outer Join 으로도 불린다.
최상단 그림 예제의 우측 상단 두번째에 해당한다.
FROM 절 아래에 RIGHT JOIN table alias ON condition 을 기술한다.
WHERE 절에 JOIN alias.column IS NOT NULL 을 기술한다.


풀 아우터 조인

Full Outer Join 으로도 불린다.
최상단 그림 예제의 중앙에 해당한다.
FROM 절 아래에 FULL JOIN table alias ON condition 을 기술한다.
