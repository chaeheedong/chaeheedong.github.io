문자 함수


||

문자열을 이어 붙이는 역할을 합니다.


CHR(ascii)

아스키 코드를 입력해 줍니다.
CHR(10) : \n
CHR(13) : \r


LOWER

문자열을 소문자로 변환해 줍니다.


UPPER

문자열을 대문자로 변환해 줍니다.


INITCAP

첫글자를 대문자로,
이 외 글자는 소문자로 변환해 줍니다.


LTRIM

좌측의 특정 값을 제거 합니다.


RTRIM

우측의 특정 값을 제거 합니다.


TRIM

좌, 우측의 공백을 제거 합니다.


SUBSTR

특정 길이만큼 잘라 냅니다.


REPLACE

특정 값을 특정 값으로 변환해 줍니다.


ASCII

아스키 코드의 숫자로 변환해 줍니다.
