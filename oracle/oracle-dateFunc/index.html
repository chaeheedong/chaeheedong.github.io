날짜 함수


SYSDATE

DB SERVER의 시간을 반환 합니다.


NEXT_DAY

특정일 다음의 요일의 날짜를 반환 합니다.


LAST_DAY

해당 월의 말일을 반환 합니다.


ADD_MONTHS

특정 달의 +, - 달을 반환 합니다.


EXTRACT

조회한 날로부터
년, 월, 일, 시, 분, 초를 반환 합니다.


ROUND

날짜를 요일 기준으로 반올림 합니다.


TRUNC

날짜를 포멧 기준으로 버린 날짜를 반환 합니다.
TRUNC(DATE, FORMAT)
포멧 : 'YY', 'MM', 'DD', 'HH', 'MI'
기본 : 'DD'
